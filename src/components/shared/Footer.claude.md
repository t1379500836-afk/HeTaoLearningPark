# Footer.vue - 页脚组件

## 功能特性

- **品牌展示**：SVG图标 + "核桃学习站" 文字
- **口号展示**：让每个孩子爱上编程，在快乐中成长
- **快速链接**：关于我们、课程介绍、练习题库、联系客服
- **版权信息**：动态显示当前年份

---

## Props API

无 Props。

---

## Events

无自定义事件。

---

## 内部状态

| 状态 | 类型 | 说明 |
|------|------|------|
| `currentYear` | `Computed<number>` | 当前年份（动态计算） |

---

## 内容结构

```
Footer
├── Logo (SVG图标 + 文字)
├── Slogan (口号)
├── Footer Links (快速链接)
│   ├── 关于我们
│   ├── 课程介绍 (#levels)
│   ├── 练习题库 (#practice)
│   └── 联系客服
└── Copyright (© {year} 核桃编程学习资料站)
```
